# SBATCH partition templates

This folder contains basic sbatch templates for all the partitions available on tempest. The purpose of these templates is to provide a quick starting point with the correct SBATCH parameters for each partition.

## Templates
- [priority](priority.sbatch)
- [unsafe](unsafe.sbatch)
- [test](test.sbatch)
- [gpupriority](gpupriority.sbatch)
- [gpuunsafe](gpuunsafe.sbatch)
- [gputest](gputest.sbatch)
- [legacy](legacy.sbatch)
- [nextgen](nextgen.sbatch)
- [nextgen-long](nextgen-long.sbatch)
- [nextgen-gpu](nextgen-gpu.sbatch)
- [nextgen-gpu-long](nextgen-gpu-long.sbatch)
